### 1. Get information


#### 1.1 Brief Description


This use case allows the user to open a dialog window and check the rules of the game Dont get Angry. After clicking on the button with text "Menu" and then "Help", a dialog is opened. After pressing  the button with text "Ok" the dialog is closed and the game can start/ continue.
